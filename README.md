# Asterisk Web Port

This project is a browser-based, single-player port of the WinForms version of **Asterisk** in one self-contained `index.html` file.

## How to Play
- Open `index.html` in a modern browser.
- Hold **Enter** to rise.
- Release **Enter** to fall.
- Reach the gap on the far right to advance.
- Hit a wall, obstacle, or the top/bottom of the playfield and the run ends.
- Press **R** to restart.

## Included in the base version
- Canvas-based gameplay
- Single-player controls
- 2×2 obstacle blocks
- Increasing difficulty
- Countdown
- Score and level display
- Persistent high score with `localStorage`
